# Retail Performance Dashboard (Excel)

## 📊 Project Overview
This project analyzes retail sales data to evaluate revenue, profit, and regional performance using Excel.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- SUMIFS
- Data Visualization

## 📌 Key Features
- KPI tracking (Total Revenue, Profit, AOV)
- Interactive dashboard using slicers
- Scenario analysis for pricing impact

## 🔍 Insights
- Identified top-performing region contributing highest revenue
- Observed sales trends across different time periods
